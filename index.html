Projects

 BigCoin 

A small scale blockchain for the purpose of teaching blockchain core fundamentals. The blockchain is secured through the SHA-256 hash algorithm and uses Elliptic Curve Digital Signatures on the secp256k1 elliptic curve (the same technology stack used to secure Bitcoin). The protocol also generates valid Bitcoin public / private addresses.

 Blockchain And Cryptocurrencies: A Data Science Tutorial 

A data science tutorial to walk readers through the data science life cycle while also teaching them about the technologies behind cryptocurrencies like Bitcoin, Ethereum, and Ripple. A webscraper is used to gather cryptocurrency price data from the internet which is then used to develop a machine learning model using linear regression.

 Decentralized Crowdfunding Web / Mobile Application 

A scalable decentralized platform that gives users an efficient peer-to-peer crowdfunding experience. The application is secured through smart contracts deployed over the Ethereum network. Built in Solidity, JavaScript, Web3, and Swift.
